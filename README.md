# mnist-vgg16-digit-classification
A VGG-16 inspired Convolutional Neural Network (CNN) for handwritten digit classification on the MNIST dataset using Python, TensorFlow/Keras, and OpenCV.

## Features

- MNIST handwritten digit classification
- 80% training and 20% testing split
- VGG-16 inspired CNN architecture
- TensorFlow/Keras implementation
- OpenCV for image processing
- Accuracy and loss visualization
- Model prediction on sample images

## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## Dataset

- MNIST Handwritten Digits
- 70,000 grayscale images
- 10 classes (0–9)

## Project Workflow

1. Load MNIST dataset
2. Split dataset (80% training, 20% testing)
3. Preprocess images
4. Build a VGG-16 inspired CNN
5. Train the model
6. Evaluate model performance
7. Predict handwritten digits

## Results

- High classification accuracy on the MNIST test dataset
- Training and validation accuracy/loss graphs
- Sample digit prediction

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python vgg16_mnist_project.py
```
